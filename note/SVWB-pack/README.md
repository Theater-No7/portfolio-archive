**シャドウバース: ワールドビヨンド**（シャドバ WB、Shadowverse: Worlds Beyond）の**カードパックの概要と、パックを引くための「ルピ」を初心者でも簡単に稼ぐ方法を解説します！**

[シャドバ WB公式サイトはこちら](https://shadowverse-wb.com/ja/)

-   **中身**：カード 8枚
    
-   **レジェンド確定**：10パック引くごとに、最高レアの「レジェンドカード」が**必ず1枚**出現します！✨

-   途中でレジェンドカードを引くと、確定までのカウントはリセットされます。

## １．招待コードの入力【所要時間2分：2,000ルピ】

シャドバ WBでは、**「招待コード」**を入力するだけで**2,000ルピ**を即時にゲットできます。

招待コードはシャドバ WB に他のプレイヤーを招待するためのコードで、招待した側、された側の双方に報酬がもらえます！

### 招待コードの入力方法

1．ホーム画面右上のメニューボタンを押します

[![画像](https://assets.st-note.com/img/1755398114-hEIocaMzOvLCJf0Qr4m3TukD.png?width=1200)](https://assets.st-note.com/img/1755398114-hEIocaMzOvLCJf0Qr4m3TukD.png?width=2000&height=2000&fit=bounds&quality=85)

シャドバ WBのホーム画面右上、右上のメニューボタンを押す

2．「友達招待」というボタンを押します

[![画像](https://assets.st-note.com/img/1755398120-t3WNb6c2dlREon1MpUJFjfYa.png?width=1200)](https://assets.st-note.com/img/1755398120-t3WNb6c2dlREon1MpUJFjfYa.png?width=2000&height=2000&fit=bounds&quality=85)

シャドバ WBのメニュー、「友達招待」ボタンを押す

3．招待コードの入力欄に招待コードを入力してください  
※ 以下の画面では入力済みの画面です。

[![画像](https://assets.st-note.com/img/1755398125-VGz7KCLfixBtcMpsu1vHZAQW.png?width=1200)](https://assets.st-note.com/img/1755398125-VGz7KCLfixBtcMpsu1vHZAQW.png?width=2000&height=2000&fit=bounds&quality=85)

## ２．Cygames iD連携【所要時間3分：2,500ルピ分】

Cygames iDとは、**Cygames が提供している他のゲーム・サービスを利用するための共通アカウントのこと**です。  
Cygames iDを連携することでゲームデータの引き継ぎが簡単になったり、公式のWeb ストアで限定商品を購入できるようになります。  
Cygames iDの詳細は以下のサイトをご覧ください。

[**Cygames ID (サイゲームスID) | Cygames** _Cygames ID でつながろう。Cygames ID (サイゲームスID) とは、Cygamesが提供する各種サービス_ _cygames.com_](https://cygames.com/)[](https://cygames.com/)

**iDを連携すると、さらに追加特典として、カードパック「伝説の幕開け」を引けるチケットを5枚もらうことができます**。

## ４．バトルガイド

ソロプレイモードの「バトルガイド」は、**ゲームの遊び方を網羅的に学べる上に報酬も美味しいのでぜひクリアしておきましょう！**

バトルガイドには、以下の2つのモードが用意されています。

-   **バトルチュートリアル**：実際のバトルに沿ったチュートリアル
    
-   **パズルレッスン**：詰め将棋のような、特定の盤面の対処法などを学べる
    

### バトルチュートリアル【所要時間60分：3,200ルピ】

## ５．プラクティスバトル

ソロプレイモードの「プラクティスバトル」は、**各クラスのデッキを使うCPUと対戦できる、実践的な練習モードです**。

[![画像](https://assets.st-note.com/img/1755400078-Txbd3V5CFnkzGcrNQBMAlogv.png?width=1200)](https://assets.st-note.com/img/1755400078-Txbd3V5CFnkzGcrNQBMAlogv.png?width=2000&height=2000&fit=bounds&quality=85)

シャドバ WBのプラクティスバトル画面

プラクティスバトルには、以下の3つの難易度が用意されています。

-   **初級（7クラス分、1クラスクリアで50ルピ）**
    
-   **中級（7クラス分、1クラスクリアで100ルピ）**
    
-   **上級（7クラス分、1クラスクリアで200ルピ）**
    

筆者の所感ですが、初級、中級は正直簡単なのでそれなりにデッキが構築できていれば勝てます。ただし上級は意外と難易度が高いので油断禁物です。

### 初級【所要時間30分：350ルピ】

初級では、**CPUのリーダーの体力が最初から「10」で始まります。**  
決して強くない上、試合時間も短いのでサクッとクリアできるはずです。

### 中級【所要時間50分：700ルピ】

中級では、**CPUのリーダーの体力が最初から「15」で始まります。**  
初級ほどではありませんが、比較的難易度は低く、状況によってはサクサク進められるはずです。

### 上級【所要時間??分：1,400ルピ】（デッキの強さや相性によります）

上級では、**CPUのリーダーの体力が最初から「20」、つまり通常のバトルと同じ状況で始まります。**  
デッキが十分に構築されていれば戦うことはできますが、油断しているといつの間にか不利な状況になっていることもあるので注意が必要です。

筆者は「ウィッチ上級」に全く勝てず、体感20回くらい挑んだ気がします…。  
**クラスの相性などを把握して、慎重に挑むのが大切かもしれません！**

## 【毎日・毎週】コツコツ稼ぐ方法

**上記の方法でもらえる報酬はすべて初回達成時のみ受け取ることができます！  
**この章では、毎日、または毎週受け取ることのできる報酬で、ルピをコツコツ稼ぐ方法を紹介します。

### ログインボーナス【毎日：500ルピ分】

[![画像](https://assets.st-note.com/img/1755407779-oDCZbRsq71IEgOVHlUmNk9cj.png?width=1200)](https://assets.st-note.com/img/1755407779-oDCZbRsq71IEgOVHlUmNk9cj.png?width=2000&height=2000&fit=bounds&quality=85)

シャドバ WBのカードパック（ガチャ）画面

シャドバ WB は、ログインボーナスとして、**カードパックを引けるチケットを毎日1枚もらえます**。  
現在は第2弾パックの「インフィニティ・エボルブ」のチケットがもらえるので、おそらくその時の最新弾のチケットがもらえるのかなと思います。  
（第3弾パックは8月下旬追加予定とのことです！）

### デイリーミッション【毎日：210ルピ〜】

[![画像](https://assets.st-note.com/img/1755408234-pIQM9yAv5CHsE12uRbnNdlS7.png?width=1200)](https://assets.st-note.com/img/1755408234-pIQM9yAv5CHsE12uRbnNdlS7.png?width=2000&height=2000&fit=bounds&quality=85)

シャドバ WBのデイリーミッション画面

**毎日更新されるデイリーミッションをクリアすることでルピを稼ぐことができます**。  
デイリーミッションは、クリアしてから21時間後に更新されます。

### パークミッション【毎週：250ルピ〜】

[![画像](https://assets.st-note.com/img/1755409029-E5dfmNq64leZSDrhM1cpCoPB.png?width=1200)](https://assets.st-note.com/img/1755409029-E5dfmNq64leZSDrhM1cpCoPB.png?width=2000&height=2000&fit=bounds&quality=85)

シャドバ WBのパークミッション画面

**シャドバパーク内のミッションクリア（80pt獲得）でも250ルピを受け取れます**。

ミッションをクリアしていくと「トレジャーキー」を受け取ることができます。その「トレジャーキー」を消費して「パークトレジャー」を引くことでもルピが手に入りますが、**運が良いと「パークトレジャー」からレジェンドカードを引けることもある**ので「トレジャーキー」はたくさん集めましょう。

### アチーブメント【累計】

[![画像](https://assets.st-note.com/img/1755408699-cJ8I0C4AtkjfsdNeUaiO57Wn.png?width=1200)](https://assets.st-note.com/img/1755408699-cJ8I0C4AtkjfsdNeUaiO57Wn.png?width=2000&height=2000&fit=bounds&quality=85)
